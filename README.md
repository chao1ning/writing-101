# **Installation**
## **Prerequisites:**
Drivers for your CNS, PNS, bones, eyes, skeletal muscles are required. Also install the Cyrillic alphabet via the following command(for ubuntu, for other operating systems please see github.com/Slavic/cyrillic/README.md)
`git clone https://github.com/Slavic/cyrillic
cd cyrillic
chmod +x ./cyrillic.gov
./cyrillic.gov`
